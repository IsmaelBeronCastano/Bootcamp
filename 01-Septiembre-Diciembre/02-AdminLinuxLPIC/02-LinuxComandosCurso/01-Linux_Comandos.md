# LINUX COMANDOS - Sistema de ficheros


## Ayuda en la terminal

- Para buscar ayuda dispongo de

> man comando
> info comando
> locate elemento (para ubicar un elemento)
> /usr/share/doc
> -h (--help, para refrescar la memoria)

- Hay otros:
  - whatis
  - apropos
  - whereis

## Navegar por los directorios

> cd
